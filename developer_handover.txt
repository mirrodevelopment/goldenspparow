Version: 1.0

# 1. OVERVIEW

Golden Sparrow provides a complete interactive 3D destination discovery globe and a static destination data API.

The Flutter application is responsible for:

* Loading the globe inside a WebView
* Listening for destination selection events
* Reading destination data from the JSON API
* Displaying destination information in the app UI

Golden Sparrow does not provide:

* Flutter UI
* Navigation
* Search UI
* Bookmarks
* Booking Flow

These are implemented by the Flutter application.

# 4. DESTINATION SELECTION EVENT

When a destination is selected on the globe, Golden Sparrow sends:

```json
{
  "type": "destination_selected",
  "destinationId": "japan"
}
```

Example:

```json
{
  "type": "destination_selected",
  "destinationId": "norway"
}
```

The destinationId always matches a key inside:

```text
https://mirro.in/api/destinations.json
```

# 7. WEBVIEW IMPLEMENTATION

```dart
import 'package:flutter/material.dart';
import 'package:webview_flutter/webview_flutter.dart';

class GlobeScreen extends StatefulWidget {
  const GlobeScreen({super.key});

  @override
  State<GlobeScreen> createState() => _GlobeScreenState();
}

class _GlobeScreenState extends State<GlobeScreen> {

  late final WebViewController controller;

  @override
  void initState() {
    super.initState();

    controller = WebViewController()
      ..setJavaScriptMode(
        JavaScriptMode.unrestricted,
      )
      ..loadRequest(
        Uri.parse(
          'https://mirro.in/goldensparrow.html',
        ),
      );
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: WebViewWidget(
        controller: controller,
      ),
    );
  }
}
```

# 8. RECEIVING DESTINATION EVENTS

Flutter must listen for messages from the WebView.

Expected Message:

```json
{
  "type": "destination_selected",
  "destinationId": "japan"
}
```

Example Handler:

```dart
void onDestinationSelected(
  String destinationId,
) async {

  print(destinationId);

  await loadDestination(
    destinationId,
  );
}
```

---

# 9. FETCHING DESTINATION DATA

```dart
import 'dart:convert';
import 'package:http/http.dart' as http;

Future<void> loadDestination(
  String destinationId,
) async {

  final response = await http.get(
    Uri.parse(
      'https://mirro.in/api/destinations.json',
    ),
  );

  final destinations =
      jsonDecode(response.body);

  final destination =
      destinations[destinationId];

  print(destination);
}
```

# 13. SAMPLE IMPLEMENTATION

```dart
Future<void> loadDestination(
  String destinationId,
) async {

  final response = await http.get(
    Uri.parse(
      'https://mirro.in/api/destinations.json',
    ),
  );

  final data =
      jsonDecode(response.body);

  final destination =
      data[destinationId];

  print(
      destination["country"]);

  print(
      destination["seasonName"]);

  print(
      destination["heroImage"]);
}
```
